## IP Log Analyzer

This project is a Python-based log analysis tool that extracts IP addresses from log files, 
tracks activity, and detects suspicious behavior such as repeated failed login attempts and potential brute force attacks. 
It identifies how often each IP appears and highlights suspicious sources based on failed login patterns. 

The project demonstrates basic cybersecurity concepts like log analysis, pattern recognition, and threat detection, 
providing a simple example of how security analysts monitor and analyze system activity.
